# IELTS Exam Platform - Frontend

Production-ready React frontend for three-role examination system.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Features

- Three-role authentication (Owner, Admin, Student)
- Modern UI with Tailwind CSS and Framer Motion
- Dark/Light mode support
- Responsive design
- File upload and parsing (JSON, XLSX, CSV)
- Test creation with multi-step form
- Real-time exam interface
- Text highlighting
- Auto-save functionality
- Answer sheets
- Charts and analytics
